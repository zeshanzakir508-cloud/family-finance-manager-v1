# Accounts Feature

A comprehensive account management system for tracking financial accounts.

## Overview

The Accounts feature provides complete management of financial accounts, enabling users to:

- **Create accounts** with name, description, opening balance, icon, and color
- **Edit accounts** to update details
- **Archive accounts** to hide them from active lists
- **Restore archived accounts** when needed
- **Delete accounts** (only if balance is zero)
- **Transfer money** between accounts
- **Search and filter** accounts by name and status
- **Track balances** with real-time updates
