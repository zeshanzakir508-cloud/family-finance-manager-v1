# Categories Feature

A comprehensive category management system for organizing transactions.

## Overview

The Categories feature provides complete management of transaction categories, enabling users to:

- **Create categories** with name, description, icon, and color
- **Edit categories** to update details
- **Archive categories** to hide them from active lists
- **Restore archived categories** when needed
- **Delete categories** (only if not in use)
- **Merge categories** to consolidate transactions
- **Search and filter** categories by name and status
