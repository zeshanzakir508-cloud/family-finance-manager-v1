# Onboarding Feature

A comprehensive onboarding flow for new users to set up their family budget application.

## Overview

The Onboarding feature provides a guided setup process that walks new users through:
- Welcome introduction
- Permission requests
- Terms and privacy policy acceptance
- Family setup (create or join)
- Account creation
- Category selection
- Completion celebration
