# Transactions Feature

A comprehensive transaction management system for tracking financial transactions.

## Overview

The Transactions feature provides complete management of financial transactions, enabling users to:

- **Create transactions** with amount, date, type, account, and category
- **Edit transactions** to update details
- **Delete transactions** with confirmation
- **Duplicate transactions** for quick entry
- **Filter transactions** by type, account, category, and status
- **Sort transactions** by date, amount, and category
- **Search transactions** by description
- **Recurring transactions** with customizable frequency
- **View transaction details** with full information
